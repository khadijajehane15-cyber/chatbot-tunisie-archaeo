\# Rapport - Chatbot RAG



\## 1. Introduction

Projet de chatbot RAG sur les sites archéologiques tunisiens.



\## 2. Architecture

\- LLM : Gemma3:1b via Ollama

\- Base vectorielle : ChromaDB

\- Embeddings : all-MiniLM-L6-v2

\- Interface : Streamlit



\## 3. Difficultés

\- Mémoire GPU limitée → Gemma3:1b

\- Configuration Ollama → Mode CPU



\## 4. Résultats

Chatbot fonctionnel avec 54 documents, réponses sourcées.
