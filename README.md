\# Chatbot RAG - Sites Archéologiques de Tunisie



\## Description

Chatbot intelligent spécialisé sur les sites archéologiques tunisiens utilisant l'architecture RAG.



\## Installation

1\. Python 3.10+

2\. `pip install -r requirements.txt`

3\. Ollama : `ollama pull gemma3:1b`



\## Utilisation

\- Créer base : `python ingest.py`

\- Tester : `python rag.py`

\- Interface : `streamlit run app.py`



\## Données

54 documents sur 10 sites : Carthage, Dougga, El Jem, Kerkouane, Bulla Regia, Sbeitla, Utique, Thuburbo Majus, Makthar, Chemtou
